Navigation Button

CSS selector:

.button-nav

Sample code:

<a class="button-nav" href="#">Navigation Button</a>

Rendered element:

Navigation Button

In-text link

CSS selector:

a

Sample code:

<a href="#">Title of link</a>

Rendered element:

Title of link

Submit/Reset Buttons

CSS selector:

.submit-reset

Sample code:

<a class="button-nav" href="#">

Rendered element:

Submit

Text Input Field

CSS selector:

input type="text"

Sample code:

<input type="text">

Rendered element:

Headings

CSS selector:

h1, h2, h3

Sample code:

<h1>Header 1</h1> <h2>Header 2</h2> <h3>Header 3</h3>

Rendered element:

Header 1

Header 2

Header 3

Paragraphs

CSS selector:

p

Sample code:

<p>This is a paragraph</p>

Rendered element:

This is a paragraph

Bulleted Lists

CSS selector:

ul

Sample code:

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Rendered element:

  • Item 1
  • Item 2
  • Item 3

Numbered Lists

CSS selector:

ol

Sample code:

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

Rendered element:

  1. Item 1
  2. Item 2
  3. Item 3
